# Кейс 4 — Интерактивный калькулятор

Учебный проект по практике.

## Описание
Веб-калькулятор на HTML, CSS и JavaScript.

Реализовано:
- 2 поля ввода чисел
- 4 операции: сложение, вычитание, умножение, деление
- Вывод результата на страницу
- Проверка некорректного ввода
- Обработка деления на ноль

## Как запустить
Открыть файл `index.html` в браузере.
